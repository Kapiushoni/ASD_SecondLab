# 🚀 Linear Data Structures Analysis

Цей проєкт присвячений реалізації та порівняльному аналізу базових лінійних структур даних на мові **C++**. Робота виконана в межах навчального курсу кафедри **ІПСА, КПІ**.

## 📋 Огляд проєкту

Проєкт містить власну реалізацію двох типів структур даних для зберігання об'єктів `MathLesson` (структура за варіантом №5):
1. **Dynamic Array** (Динамічний масив з автоматичним розширенням `capacity`).
2. **Singly Linked List** (Однозв’язний список).

### Ключові можливості:
- Повна реалізація операцій `push/pop` (front & back), `get`, `insert`, `remove`.
- Аналіз амортизованої складності динамічного масиву.
- Порівняльне тестування швидкодії на великих обсягах даних (до 50,000 операцій).
- Додаткові алгоритми: перевірка дужкової послідовності (Stack), розворот списку та пошук циклу (Floyd's Cycle-finding algorithm).

## 📊 Порівняльний аналіз (Asymptotic Complexity)

| Операція             | Dynamic Array | Linked List |
|----------------------|---------------|-------------|
| **Access (get)** | $O(1)$        | $O(n)$      |
| **Insert/Delete Front**| $O(n)$        | $O(1)$      |
| **Insert Back** | $O(1)^*$      | $O(n)$      |
| **Delete Back** | $O(1)$        | $O(n)$      |

*\* Амортизована складність при розширенні масиву у 2 рази.*
